# Recruitment CRM Workflow

## Main Workflow

1. Company is added to the CRM.
2. Company creates a job posting.
3. Candidate is added to the CRM.
4. Candidate applies to a job.
5. Application is created to connect the candidate with the job.
6. Application moves through hiring stages:
   - Applied
   - Screening
   - Interview Scheduled
   - Interview Done
   - Offer Sent
   - Hired
   - Rejected
7. Interview is scheduled for the application.
8. Evaluation is added after screening or interview.
9. Activity/reminder is created for follow-up tasks.

## Example

EVA Ops creates a Backend Developer job.

John Smith applies to the Backend Developer job.

An application is created:
John Smith → Backend Developer → Screening

Then the recruiter can schedule an interview, add evaluation, and create follow-up activities.
