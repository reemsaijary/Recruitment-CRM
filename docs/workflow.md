# Recruitment CRM Workflow

1. Company posts a job.
2. Candidate applies to the job.
3. Application is created.
4. Application moves through stages:
   - Applied
   - Screening
   - Interview Scheduled
   - Interview Done
   - Offer Sent
   - Hired
   - Rejected
5. Interview can be scheduled for an application.
6. Evaluation can be added after screening or interview.
