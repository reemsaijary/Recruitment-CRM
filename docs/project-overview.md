# Recruitment CRM

## Project Idea
A recruitment CRM and ATS platform designed for recruitment agencies and companies to manage jobs, candidates, applications, interviews, and hiring workflows.

## Main Features
- Company profiles
- Candidate profiles
- Job postings
- Candidate applications
- Interview scheduling
- Recruitment workflow tracking
- Activity and follow-up management

## Main Profiles
1. Companies
2. Candidates

## Goal
Build the backend foundation first using Django and Supabase before starting the frontend UI.
