# Recruitment CRM

## Overview

Recruitment CRM is a full-stack recruitment management platform built with Django.

The system helps administrators, companies, and candidates manage the complete recruitment lifecycle, from job creation and candidate applications to interviews, evaluations, hiring decisions, and workflow tracking.

The platform was developed as part of an internship project and later expanded with modern UI design, authentication, notifications, deployment, and role-based dashboards.

---

## Core Features

### Authentication & User Management

- Role-based authentication
- Admin accounts
- Company accounts
- Candidate accounts
- Google OAuth login for candidates
- Secure access control

---

## Admin Portal

Administrators can:

- Manage companies
- Manage candidates
- Manage jobs
- Manage applications
- Manage interviews
- Manage evaluations
- Manage activities
- Monitor platform statistics

---

## Company Portal

Companies can:

- Create job postings
- Edit and manage jobs
- Review applications
- Update application statuses
- Schedule interviews
- Track candidates through a Kanban board
- Manage hiring workflows

---

## Candidate Portal

Candidates can:

- Register and login
- Login using Google
- Browse available jobs
- Apply to jobs
- Upload CVs
- Manage profiles
- Track application progress

---

## Notification System

The platform includes a notification system that automatically informs users when:

- A candidate applies to a job
- An interview is scheduled
- An application status changes

---
